# Customer-Churn-Prediction
Machine learning project to predict telecom customer churn using multiple classification models.

## Dataset
- Telco Customer Churn Dataset
- 7,043 customer records
- Features include tenure, contract type, payment method, internet service, and charges

## Models Implemented
- Logistic Regression
- Decision Tree
- Random Forest
- XGBoost

## Best Performing Model
Logistic Regression achieved the best overall performance based on Accuracy, F1-score, and ROC–AUC.

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- ROC–AUC
- Confusion Matrix

## Visualizations
- EDA plots
- Logistic regression predictive curve
- Confusion matrices
- ROC curves
- Model comparison bar graphs

## Dashboard
A simple Streamlit dashboard is included to visualize churn patterns and model insights.
